# 100x-dev

Since you already are a 10x dev, why not go one step further and be a 100x dev.

## Features
This extension provides 2 functions to call via command palette.
### 100x Transform
Become 100x instantly

### 100x Revert
Revert back to uga booga dev form

## Requirements
Dev should have more than 2 brain cells to use this extension.

## Extension Settings
You can set your power level and boost level to go even further.

For example (in Settings JSON):
* `1000x.powerLevel`: 20000,
* `1000x.boostLevel`: 40000,
